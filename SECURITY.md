# Security Policy

## Supported Versions

| Branch  | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| dev     | :white_check_mark: |

## Overview

Polydock Engine is a running server that orchestrates Lagoon instances. It handles sensitive data including:

- **SSH keys** used for Lagoon communication
- **User data** for authenticated users and service accounts

We take the security of this system seriously.

## Reporting a Vulnerability

We ask that all suspected vulnerabilities be privately and responsibly disclosed by [contacting our security team](mailto:security+polydock@amazee.io) or by using the private vulnerability reporting form at [amazeeio/polydock-engine/security](https://github.com/amazeeio/polydock-engine/security).

## Communication

[GitHub Security Advisories](https://github.com/amazeeio/polydock-engine/security/advisories) will be used to communicate during the process of identifying and resolving the vulnerability.

The advisory will only be made public when the patched version is released to inform the community of the vulnerability and its potential security impact.
